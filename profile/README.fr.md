# AbsyssLab

[🇬🇧 English version](README.md)

Bienvenue sur le profil GitHub d'**Absyss** !

## 🎯 Notre Mission

Nous développons des **connecteurs open source** pour **Visual TOM** (VTOM), une solution leader d'orchestration et d'automatisation des workflows IT. Notre objectif est de faciliter l'intégration de VTOM avec l'écosystème moderne des outils DevOps, Cloud et Data.

## 🔌 Nos Connecteurs

Découvrez notre collection complète de connecteurs VTOM :

### ☁️ Cloud & Automation
- **[vtom-azure-data-factory](https://github.com/AbsyssLab/vtom-azure-data-factory)** - Orchestration des pipelines Azure Data Factory
- **[vtom-azure-logic-app](https://github.com/AbsyssLab/vtom-azure-logic-app)** - Déclenchement de workflows Azure Logic Apps
- **[vtom-azure-storage](https://github.com/AbsyssLab/vtom-azure-storage)** - Gestion du stockage Azure
- **[vtom-ansible](https://github.com/AbsyssLab/vtom-ansible)** - Exécution de playbooks Ansible
- **[vtom-jenkins](https://github.com/AbsyssLab/vtom-jenkins)** - Intégration CI/CD avec Jenkins

### 🖥️ Virtualisation & Infrastructure
- **[vtom-vmware](https://github.com/AbsyssLab/vtom-vmware)** - Gestion et orchestration des VMs VMware vSphere

### 💾 Backup & Recovery
- **[vtom-veeam](https://github.com/AbsyssLab/vtom-veeam)** - Orchestration des sauvegardes Veeam
- **[vtom-commvault](https://github.com/AbsyssLab/vtom-commvault)** - Gestion des jobs Commvault
- **[vtom-hycu](https://github.com/AbsyssLab/vtom-hycu)** - Intégration avec Hycu

### 📊 Monitoring & Alerting
- **[vtom-servicenow](https://github.com/AbsyssLab/vtom-servicenow)** - Création de tickets ServiceNow
- **[vtom-jira-service-management](https://github.com/AbsyssLab/vtom-jira-service-management)** - Intégration avec Jira Service Management
- **[vtom-pager-duty](https://github.com/AbsyssLab/vtom-pager-duty)** - Gestion des alertes PagerDuty
- **[vtom-grafana](https://github.com/AbsyssLab/vtom-grafana)** - Dashboards Grafana pour VTOM

### 🤖 AI & Innovation
- **[vtom-ai-jobs](https://github.com/AbsyssLab/vtom-ai-jobs)** - Exécution de jobs avec prompts IA
- **[vtom-ai-log-analyzer](https://github.com/AbsyssLab/vtom-ai-log-analyzer)** - Analyse intelligente des logs d'alarmes

### 🛠️ DevOps & Management
- **[vtom-git](https://github.com/AbsyssLab/vtom-git)** - Gestion des scripts dans Git
- **[vtom-gitcommand](https://github.com/AbsyssLab/vtom-gitcommand)** - Commandes Git avancées
- **[vtom-jobascode](https://github.com/AbsyssLab/vtom-jobascode)** - Scheduling as Code
- **[vtom-remote-agent-unix](https://github.com/AbsyssLab/vtom-remote-agent-unix)** - Configuration des agents distants Unix
